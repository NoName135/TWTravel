## 功能

測試帳號密碼

```bash
帳號： qq@qq.qq
密碼： 11111111
```

- [x] 登入/登出
- [x] 景點搜尋
- [x] 景點詳細資料
- [x] 天氣預報
- [x] 收藏景點
- [x] 建立行程
- [x] 編輯行程
- [x] 變更行程日期時間
- [x] 自動計算行程時間
- [x] 輸入地址自動計算時間
- [x] 拖曳變更行程 Day
- [x] 拖曳變更行程地點
- [x] 列印當日行程
- [ ] 測速照相功能(來源未開啟 CORS 無法介接)
